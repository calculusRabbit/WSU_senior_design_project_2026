# WSU_senior_design_project_2026
Smart Campus Assistant Platform

# Overview
The Smart Campus Assistant Platform is a web-based intelligent system designed to enhance student  access to campus information at university. Students frequently encounter difficulties locating accurate  and timely information due to fragmented systems and information overload. This project aims to  centralize essential campus resources and provide personalized assistance through a single, easy-to-use  platform. The system will feature an AI-powered chatbot that allows students to ask natural language questions  related to campus services, academic schedules, building locations, deadlines, and events. By  combining structured campus data with intelligent query processing, the platform will deliver fast,  relevant responses.  Additional functionality such as class schedule management, interactive campus navigation, and  targeted notifications will further improve the student experience. The system will fetch data from the Blackboard LMS via REST APIs; data will be fetched from course scheduling/registration systems via  web scraping. The app will also provide information about dining options on campus. The app will  provide recommendations to plan studying and course attendance (for instance, recommend attending a  course in which attendance is tracked on BlackBoard, suggesting study day/week plans for exams and  quizzes, etc.). Using a RateMyProfessors API to show course/instructor review and student feedback about the  course. The app will automatically create a group chat for each class, allowing students to collaborate,  share notes, and discuss; the chat will also have a “find study partner” option that would only notify those available during selected study times without using other chat apps.
